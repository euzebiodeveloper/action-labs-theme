# Action Labs Theme

Action Labs Theme is a lightweight WordPress theme created as part of a job selection test. It is a focused, minimal starter theme intended to demonstrate front-end structure, theme features and a simple release workflow.

## Sobre este tema

- Propósito: teste para processo seletivo — demonstrar habilidades em PHP/WordPress, HTML, CSS e automações de build.
- Estrutura limpa e arquivos organizados em `inc/`, `template-parts/`, `assets/` e `js/`.
- Suporta logo customizada (exibida quando configurada) e layouts responsivos básicos.

## Como usar

1. Copie esta pasta para `wp-content/themes/` e ative o tema no painel do WordPress.
2. Configure a identidade do site em Aparência → Personalizar → Identidade do site (para adicionar logo e título).
3. Edite páginas pelo editor padrão do WordPress; o template de página aplica padding lateral e cor do título conforme projeto.

## Regras de estilo e comportamento

- Título das páginas: cor padrão `#01758D`.
- Padding das páginas: `120px` lateral em desktop; `16px` em telas com largura ≤ 600px.
- Nos cards de listagem, somente o botão "LEIA MAIS" é link — imagem e título não são clicáveis.

## Release / ZIP

O repositório inclui uma versão em formato .zip disponível para realizar o teste em seu ambiente Wordpress.

## Observações finais

Este tema foi criado para um teste de seleção — ele não é um produto final. Use-o como base, modifique e expanda conforme a necessidade para projetos reais.

Se quiser, posso ajudar a gerar recursos adicionais (documentação de instalação, imagens de demonstração, ou ajustar o workflow de release).
